# progetto-mdc2026

## Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni

```bash
git clone "https://github.com/tom-acciarresi/progetto-mgc2026.git";
cd progetto-mgc2026 &&
gradle run
```

### Controlli

- W, A, S, D -- Movimento
- ESC -- Esci dal gioco
