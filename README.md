# Sword Game

A "simple" 2D Zelda-like game

<img src="images/game_screenshot.png" width="1920">

## Prerequisites for running the game
- Java 25
- Gradle

## How to run the game

```bash
git clone "https://github.com/tom-acciarresi/progetto-mgc2026.git";
cd progetto-mgc2026 &&
gradle run
```

## Controls

- W, A, S, D -- Movement
- ⬅, ⬆, ➡, ⬇ -- Directional attack
- ESC -- Quit
