# Sword Game

A "simple" 2D Zelda-like game

<img src="images/game_screenshot.png" width="1920">

## Prerequisites for running the game
- Java 25
- Gradle

## How to run the game

Clone repo
```bash
git clone "https://github.com/tom-acciarresi/sword-game.git"
```

Run game
```bash
cd sword-game &&
gradle run
```

## Controls

- W, A, S, D -- Movement
- ⬅, ⬆, ➡, ⬇ -- Directional attack
- ESC -- Quit
