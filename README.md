# progetto-mdc2026

## Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni

```bash
git clone <url-del-repository>
cd <nome-cartella>
```

### Build del progetto

```bash
./gradlew build
```

### Esecuzione

```bash
./gradlew run
```
