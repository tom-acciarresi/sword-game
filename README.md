# Sword Game

A "simple" 2D Zelda-like game

<img src="images/game_screenshot.png" width="1920">

## Prerequisites for running the game
- Java 25 or newer

## How to run the game

### 1. Download the repository in whatever way you wish

```bash
git clone "https://github.com/tom-acciarresi/sword-game.git"
```

OR

Just download and unzip from github

### 2. Change working directory to the game's directory
```bash
cd sword-game
```

### 3. Run the game

Run game on Windows
```bash
.\gradlew.bat run
```

OR

Run game on Linux:
```bash
./gradlew run
```

## Controls

- W, A, S, D -- Movement
- ⬅, ⬆, ➡, ⬇ -- Directional attack
- ESC -- Quit

## Save Location

Under the `{username}` folder in Windows, `/home/` in Linux

`$user.home/Sword_Game/sav.dat`
